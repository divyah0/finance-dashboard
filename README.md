# 💰 Finance Dashboard (Vanilla JavaScript)

## 📌 Overview
This project is a simple and interactive Finance Dashboard built using HTML, CSS, and Vanilla JavaScript. It allows users to track financial activities, visualize spending patterns, and manage transactions with a clean and intuitive UI.

---

## 🚀 Features

### 📊 Dashboard Overview
- Displays Total Balance, Income, and Expenses
- Real-time updates based on transactions

### 📈 Data Visualization
- Line chart for monthly trends
- Pie chart for category-wise breakdown
- Implemented using Chart.js

### 📋 Transactions Management
- View all transactions in a table
- Search transactions by category
- Add new transactions (Admin only)
- Delete transactions (Admin only)

### 👤 Role-Based UI
- **Viewer** → Read-only access
- **Admin** → Can add & delete transactions

### 💡 Insights Section
- Shows highest spending category dynamically

### 💾 Data Persistence
- Uses localStorage to save data
- Data remains after page refresh

### ⚠️ Smart Validation
- Prevents adding expense if balance is insufficient
- Displays alert for invalid transactions

### 🎨 UI/UX
- Clean and modern design
- Responsive layout for mobile and desktop

---

## 🛠️ Tech Stack

- HTML5
